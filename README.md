# 安装 googletest 开发工具包 (libgtest-dev)
Ubuntu 18.04
```
sudo apt-get install -y googletest libgtest-dev google-mock
```

# 安装 CMake
```
sudo apt-get install -y cmake cmake-gui
```

# 编译配置
```
cmake .
```
或
```
cmake-gui .
```

# make 编译
```
make
```

# 运行测试用例
```
ctest
```
